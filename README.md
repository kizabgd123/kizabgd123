## 🔬 Core Methodologies & Engineering Principles

My approach to Machine Learning and Software Architecture is grounded in strict, proven paradigms:

> **1. Ensembling Philosophy:** I exclusively use **Rank Blending** over simple point averaging in Kaggle ensembles to guarantee robust stabilization against outlier predictions.
> 
> **2. Categorical Handling:** Strong preference for architectures that provide native handling of categorical features (CatBoost/LightGBM) over manual preprocessing schemes (OHE/Target Encoding), allowing models to internalize category structures.
> 
> **3. Reliable AI Execution:** Operations are fortified against LLM hallucination/simulation via the **Anti-Simulation rule**—every significant state change must be physically written to disk and validated by hash before the system proceeds.
